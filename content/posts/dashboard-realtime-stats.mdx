## 배경

AI Studios v3 플랫폼에서 1000만 건 이상의 프로젝트 데이터를 실시간으로 조회할 수 있는 통계 대시보드와 데일리 리포트 자동화 시스템을 구축했습니다.

## 문제

- 대용량 데이터 조회 시 응답 지연
- 실시간성과 정확성 사이의 트레이드오ff
- 운영팀이 매일 수동으로 리포트를 만들어야 하는 비효율

## 접근

- Next.js 기반 대시보드 UI 설계
- 쿼리·캐싱 전략으로 조회 성능 개선
- 데일리 리포트 자동 생성 파이프라인 구축

## 결과

> 상세 수치와 기술 스택은 [경력기술서](/work)의 **2.1.1 대규모 데이터 실시간 통계 및 시각화** 섹션을 참고해 주세요.

문제를 숫자로 정의하고, 개선 전후를 측정하는 습관이 설계 방향을 명확하게 만들어 줬습니다.
